Rellenos


El padding es cada uno de los espacios de relleno horizontales
y verticales de un elemento.
Se aplica a todos los elementos excepto algunos elementos de
tablas como grupos de cabeceras y grupos de pies de tabla
CSS define cuatro propiedades para controlar:

padding-top:

Respecto a la parte superior

padding-right:

Respecto a la parte derecha

padding-bottom:

Respecto a la parte inferior

padding-left:

Respecto a la parte izquierda
