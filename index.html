Propiedades Border


La propiedad border permite definir en una única regla todos los
bordes de los elementos seleccionados. Se puede utilizar border para
definir el o los valores siguientes: border-width, border-style, border-color

border-block:

La propiedad de CSS border-block es una propiedad
abreviada para establecer los valores lógicos individuales del
borde de bloque en un solo lugar en la hoja de estilos.

border-block-color:

define el color del borde de bloque de un elemento,
que se asigna al color del borde físico dependiendo de los elementos
de modo de escrituro, direccionalidad y la orientación del texto. Esto
corresponde a las propiedades border-top-color y border-bottom-color, o
border-right-color (en-US) y border-left-color dependiendo de los valores
definidos para writing-mode, direction, y text-orientation.

border-color:

Establece el color de algunos o todos los bordes de los elementos

border-style:

Establece el estilo de algunos o todos los bordes de los elementos

  • none
  • dotted
  • solid
  • groove
  • inset
  • inherit
  • hidden
  • dashed
  • double
  • ridge
  • outset

border-top:

Establece algunas o todas las propiedades del borde superior de los elementos

border-right:

Establece algunas o todas las propiedades del borde derecho de los elementos

border-bottom:

Establece algunas o todas las propiedades del borde inferior de los elementos

border-left:

Establece algunas o todas las propiedades del borde izquierdo de los elementos

border-width:

Establece la anchura de algunos o todos los bordes de los elementos

border-radius:

Nos permite redondear esquinas

border-spacing:

Distancia entre las celdas adyacentes de la tabla
